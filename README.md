المميزات الرئيسية • يخمن يوزرات — مولد ذكي يقترح تشكيلات يوزر (3/4/5 أحرف).
• بايوات جاهزة — مئات العبارات المناسبة للتيك توك، إنستا، تويتر.
• شعر جاهز — أبيات قصيرة قابلة للنسخ.
• زخرفة الاسم — توليد زخارف حلوة للاسم مع نسخ سريع.
• موسيقى عربية — مجموعة مقاطع قصيرة مناسبة للبايو أو الريلز (روابط/نماذج).
• أفاتارات لملفك — صور جاهزة بجودة عالية.
• نصائح عامة — خطوات عملية لرفع التفاعل وحماية الحساب.
• واجهة سهلة وبسيطة، نسخ بنقرة، مشاركة مباشرة على السوشيال.

تحذير أخلاقي
التطبيق مخصّص للإبداع فقط — لا تستخدم ميزة "تخمين اليوزرات" لأغراض مضرة، تعقب أو اختراق حسابات. استخدمه باحترام وبما يتوافق مع قوانين المنصات.

لستة مفصّلة للمميزات (مقسمة)

1. مولد يوزرات

أوضاع: ثلاثية، رباعية، خماسية.

خيارات: أحرف فقط، أحرف+أرقام، استخدام رموز بسيطة.

فلتر لحجب كلمات مُسيئة.

زر نسخ ومشاركة.



2. بايوات جاهزة

مجموعات: رومانسية، طريفة، احترافية، اقتباسات، عربية/إنجليزية.

بحث داخل القوائم، حفظ المفضلة.



3. شعر / اقتباسات

أبيات قصيرة قابلة للنسخ.

تصنيف (حب، حُزن، تحفيز).

زر مشاركة.



4. زخرفة الاسم

مئات الأنماط الجاهزة.

تحويل تلقائي عند كتابة الاسم، معاينة قبل الحفظ.



5. موسيقى عربية

مكتبة مقتطفات موسيقية قصيرة (حقوق واضحة — استخدم مقاطع مرخّصة/خالية من حقوق).

معاينة ونسخ رابط أو اقتراح استخدام للمونتاج.



6. أفاتارات وصور للملف

قوالب صور، أيقونات، تعديل بسيط (قص/تدوير)، حفظ ومشاركة.



7. نصائح وـHow-to

خطوات أمان الحساب، تحسين بروفايل، أفضل أوقات النشر، نصائح للـ hashtag.



8. إضافات مقترحة (زِدتها لك)

مولد هاشتاقات ذكي بناءً على كلمة مفتاحية.

فحص اسم المستخدم: يخبرك إن كان اليوزر متاح عادة (تنبيه: مجرد اقتراح، لا يحاول الدخول على منصات).

قالب بايو متغير تلقائياً لعدة منصات (Copy for IG / Copy for TikTok).

ثيمات واجهة (فاتح/داكن + ألوان جاهزة).

إمكانية حفظ القوالب الخاصة بالمستخدم.




شاشات مقترحة للـ App Store listing / داخل التطبيق

1. شاشة رئيسية: شبكة بخانات — مولد يوزرات، زخرفة، بايو، شعر، موسيقى، أفاتارات، نصائح.


2. شاشة مولد اليوزرات: خيارات العدد، نوع الحروف، زر توليد.


3. شاشة معاينة الزخارف: إدخال اسم + عرض أنماط + نسخ.


4. شاشة مكتبة البايو: فئات + زر قلب للحفظ + نسخ.


5. شاشة المفضلات: تحفظ كل ما أعجبك.


6. شاشة إعدادات: تفعيل/تعطيل الإعلانات، ثيم، مشاركة التطبيق.
